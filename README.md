# 电商广告点击行为分析项目

## 项目背景
本项目基于淘宝广告点击数据，分析用户行为特征与广告投放效果，挖掘影响点击率（CTR）的关键因素。

## 技术栈
- Python（Pandas / Matplotlib / Seaborn）
- PostgreSQL（SQL分析）
- Jupyter Notebook

## 数据分析流程
1. 数据探索与质量检查
2. CTR整体分析
3. 用户画像分析（年龄 / 性别 / 消费能力）
4. 多维组合分析（年龄 + 消费能力）
5. 广告主与Campaign分析
6. 商品价格对CTR影响分析

## 核心结论
- 年龄层6和1用户CTR最高
- 高消费用户点击更理性
- 广告主和Campaign差异显著
- 商品价格与CTR呈负相关
